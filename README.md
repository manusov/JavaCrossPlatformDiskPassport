# JavaCrossPlatformDiskPassport
Mass storage device passport and SMART analysing utility.
Required binary image of device output data as input file.
Device info binary image file must be saved by external tools
(off-line mode only supported yet).

DiskPassport directory contains Java sources (NetBeans project).

BINARY_READER_DOS directory contains external tool, reader example for DOS environment (TASM).

BINARY_DUMPS directory contains binary images for some drives.

Java Runtime Environment (JRE) required.

Single JAR runs under Windows 32/64, Linux 32/64.

java -jar NAME.jar
